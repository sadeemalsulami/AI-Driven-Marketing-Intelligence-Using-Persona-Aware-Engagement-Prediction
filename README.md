# AI Marketing Engagement Prediction with Personas

## Status
🚧 Work in Progress (Graduation Project)

## Overview
This project investigates whether adding audience personas improves engagement prediction. Personas are created using Sentence-BERT embeddings and clustering techniques.

A baseline text-only model is compared with a persona-enhanced model.

## Objectives
- Build a baseline engagement prediction model
- Generate audience personas using embedding + clustering
- Compare model performance with and without personas

## Evaluation Metrics
- RMSE
- Accuracy
- F1-score

## Current Progress

- Initial research and model design
